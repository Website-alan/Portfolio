Benny Stores POS / Credit System

Project Overview

A full-featured Point-of-Sale (POS) and Credit Management software system built natively for desktop using the Python Django framework. Designed specifically to handle dynamic retail store operations, robust inventory management, complex customer credit systems (informal ledgers/khatta), and daily financial reconciliation.

Key Features

  • Ledger & CRM: Complete tracking of customer credit, vendor payouts, and automated global dashboards.
  • Inventory Management: Track target stock, low stock thresholds with automated alerts, and full stock movement logs.
  • Dynamic POS & Billing: Fast JS-powered checkout interface with partial payment accounting. Unpaid balances automatically sync to the customer's ledger.
  • Accounts & Cashflow: Advanced bucket-tracking (Cash Drawer, UPI, Bank) and enforced daily opening/closing reconciliation.
  • Automated Backups & Security: Immutable localized daily backups via customized Python launcher scripts and role-based action tracking.

Tech Stack

Python 3 Django Framework SQLite3 Vanilla JS & HTML/CSS Chart.js
