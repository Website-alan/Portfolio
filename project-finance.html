FinanceTrack (SplitMate)

A modern, frontend-heavy group finance manager engineered to solve the complexities of shared expenses, trips, and personal wallet management.

📸
Live Prototype Screenshot Development v0.8.4

Technical Architecture

FinanceTrack elegantly untangles complex group debts into optimized settlement instructions. By utilizing a custom-built Peer-to-Peer settlement algorithm, it minimizes the absolute number of transactions needed to clear group balances.

Core Features

  • Dynamic Dashboard: Global view of group expenses, active members, and visual key metric statistics.
  • Group Expense Engine: Intelligent capability to split costs across multiple participants.
  • Optimized Settlements: Mathematical algorithm reducing complex group debts into minimal peer-to-peer instructions.
  • Personal Wallet: A dedicated suite for tracking private income and net-balance strictly separate from group ledgers.

Tech Implementation

Vanilla JS (ES6+) HTML5 Core CSS Variables & Grid Algorithm Optimization
