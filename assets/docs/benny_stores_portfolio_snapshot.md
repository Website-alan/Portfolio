# Benny Stores Project Snapshot

This artifact contains the details, screenshots, and video for the **Benny Stores** project to be used in your personal portfolio.

## Project Overview
**Benny Stores Credit Management System (BSMS)** is a professional Point-of-Sale (POS) and Credit Management software built on the Django Framework. It is designed to handle retail store operations, inventory tracking, customer credit ledgers, and financial reconciliation.

---

## Key Features

1.  **Sales Dashboard**: Real-time financial metrics including daily sales, expenses, and estimated profit at a glance.
2.  **Point of Sale (POS)**: A dynamic interface for creating invoices with split payment support (Cash, Card, UPI) and automatic link to client ledgers.
3.  **Comprehensive Ledger**: Tracks parties (Customers & Vendors) with automated credit/debit balance updates.
4.  **Inventory Management**: Detailed product variant tracking with low-stock alerts and movement history.
5.  **Accounts & Reconciliation**: Daily opening/closing balance tracking to ensure cash drawers match digital records.

## Technical Stack
- **Backend**: Python 3, Django Framework
- **Database**: SQLite3
- **Frontend**: HTML5, CSS3 (Modern Glassmorphism Design), Vanilla JavaScript
- **Charting**: Chart.js for financial data visualization
- **Launcher**: Standalone Python launcher for local deployment
